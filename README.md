# Simple calculator project
Written in Kotlin

Supports:
- Consts:
- - `e` - euler constant
  - `pi` - pi constant
- Functions:
- - `sin`, `cos`, `ctg`, `tg` - trygonometric
  - `log`, `ln` - logarithms
  - `sqrt` - Square root
- Operators:
- - `+`, `-`, `/`, `*` - standard math
  - `|value|` - absolute value
  - `^` - power
- Settings:
- - Fixed precision or automatic
