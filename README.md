# Simple calculator project
Written in Kotlin

Supports:
- Consts:
- - `e` - euler constant
  - `pi` - pi constant
- Functions:
- - `sin`, `cos`, `ctg`, `tg` - trygonometric (radians)
  - `log`, `ln` - logarithms
  - `sqrt` - Square root
- Operators:
- - `+`, `-`, `/`, `*` - standard math
  - `|value|` - absolute value
  - `^` - power
  - `()` parenthasis
- Settings:
- - Fixed precision or automatic


Examples:
-  powers - `2^(3/2)`
-  functions - `sin(3/2*pi)`
-  logarithm - `log(2)(32)` - log 2 from 32 = 5
